# Diplom
Техническое задание – сайт приюта животных

Внешний вид: Страница со списком животных, можно зайти на каждый вид и почитать детально описание, посмотреть фото

Суть: клиент может зайти на сайт, просмотреть животных, которые находятся в приюте и может выбрать домашнее животное, чтоб забрать его себе 

Функционал:

1.	Страница со списком животных
2.	Добавление в базу животных администратором
3.	Возможно зайти на сайт, затем на каждого животного и просмотреть описание
4.	Поиск на сайте:
•	поиск по виду животного
•	поиск по породе
•	поиск по месту нахождения животного

5. Форма обратной связи с возможностью оставить контакт для связи

Технические инструменты:
1. язык программировани - python 3.12
2. backend - django
3.база данных postgres
